## Sum To N

#### Difficulty: `Low`

- - -

### Description

> Given the positive integer N and print summation from 1 to N

### Input

>The only one line represent one positive integer N.
>
>1<=N<=1,000,000

### Output

>Print the summation from 1 to N

### Sample I/O

| Input | Output |
| :---- | :----- |
| 3     | 6      |
| 10    | 55     |
